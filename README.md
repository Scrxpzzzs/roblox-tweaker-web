# 🚀 Roblox Tweaker - Web Version

A beautiful, zero-config web optimizer for Roblox performance settings.

**🔗 [LIVE DEMO](https://roblox-tweaker-web.streamlit.app)** ← Click here!

## ✨ Features

- 5 categories of tweaks (Basic, Modifications, Advanced, GPU, Input)
- Toggle switches & sliders for instant control
- Save settings to JSON file
- Modern dark theme
- Works on desktop, tablet, phone
- **Zero configuration needed**

## 🚀 Run Locally

```bash
pip install streamlit
streamlit run tweaker_web.py
```

Opens at `http://localhost:8501`

## 🌐 Deploy to Internet (FREE)

### Using Streamlit Cloud (1 minute):
1. Push to GitHub
2. Go to https://share.streamlit.io
3. Select your repo → Deploy
4. Get a public URL instantly ✅

See [`GITHUB_SETUP.md`](GITHUB_SETUP.md) for detailed instructions.

## 📁 Files

- `tweaker_web.py` - Main app
- `requirements.txt` - Dependencies
- `.streamlit/config.toml` - Theme config
- `GITHUB_SETUP.md` - Deployment guide

## 🎮 Tweaker Categories

- **Basic Tweaks** - Telemetry, memory, background apps, render quality
- **Modifications** - FPS unlocker, texture optimization, hyperion bypass
- **Advanced** - CPU boost, network tuning, VSync, thread count
- **GPU Tweaks** - Latency mode, shader cache, anti-aliasing, render distance
- **Input Tweaks** - Raw input, input lag, mouse accel, sensitivity

## 💾 Settings

Settings are stored in browser session. Click **"Save Settings to File"** to export as JSON.

## 🔒 Privacy

No IP exposure. Deploy to Streamlit Cloud for a permanent public URL.

## 📝 Customize

Edit `tweaker_web.py` to add/remove tweaks or change defaults. Push to GitHub for instant live updates!
